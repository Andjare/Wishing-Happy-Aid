# Wishing-Happy-Aid
<strong>Wishing Happy Aid</strong> is a small web application encoded in HTML, PHP, JS, and CSS. During the encoding we also used some PNG images and some effects to make the page attractive.</br>
So when people start sharing the page with friends and next to their friends with their friends, the page will spread and become Viral.</br>So if you want to earn from festival sessions then you can use <strong>Wishing Happy Aid</strong> script.</br></br>
<img src="http://mawlid.orgfree.com/git/Wishing-Happy-Aid.png" />
</br></br></br>

You can modify the script and change everything you want by modifying it in <code>config.php</code>
You will find some explanations in the comments in Arabic to help you understand the program variables easier

You can this use script on all national and religious occasions .. :blush:

You will find the attached pictures folder .. <pre>assets</pre>
This folder contains all images associated with the script.
You can change the images <strong>without changing their size (image dimensions)</strong>

For Adsense topic ..

You will notice the presence of only two ad units in the <code>config.php</code> file .. While four units will appear in the front of the site because each ad unit repeated twice.

Two small ad units:
You can change the small ad unit number in a file .. From here:

If the ad unit number is left blank, the two small ads will not fade on the site

Two large ad units:
You can change the large ad unit number in a file .. From here:

If the ad unit number is left blank, the two large ads will not be defeated

I advise you to start with one ad unit at the beginning in order to protect your advertising account from the closure ..
You can choose the small ad unit and after the first batch you can run the two large units.
